Все теги по оформлению текста + бонусные:

Теги < p > - для написания текста в абзацах

Теги < title > - для заголовка вкладки

Теги < hr > - для линии

Теги < h1(1-6) > - для размера текста

Теги < b(strong) > - для жирности текста

Теги < i(em) > - для курсива

Теги < s > - зачёркиванье текста

Теги < u > - нижнеё подчеркиванье

Теги < !--текст-- > - для пояснительных надписей

Чтобы поставить спец.символ используем & и текстовый номер который берем с сторонних сайтов

< h1 (1-6) > < b (strong) > < i (em) > используют поисковые системы, так что с умом применять их
